# simple_inquire_statement_handler
INQUIRE statement handler for Fortran

## Supported specifiers of `INQUIRE` statement

* `ACCESS`
* `ACTION`
* `BLANK`
* `DELIM`
* `DIRECT`
* `EXIST`
* `FORM`
* `FORMATTED`
* `NAME`
* `NAMED`
* `NEXTREC`
* `NUMBER`
* `OPENED`
* `PAD`
* `POSITION`
* `READ`
* `READWRITE`
* `RECL`
* `SEQUENTIAL`
* `UNFORMATTED`
* `WRITE`
